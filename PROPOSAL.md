# 🏛️ 壁画守护者 (Mural Guardian) - 项目提案

> 基于联邦学习的敦煌壁画智能修复与保护平台

---

## 1. 问题与机遇

### 1.1 市场痛点

| 痛点 | 现状 | 影响 |
|------|------|------|
| 壁画病害检测慢 | 依赖专家肉眼识别，4.5万㎡壁画仅有100多名修复师 | 病害发现滞后，损失不可逆 |
| 修复一致性难保证 | 修复师个人风格差异大，同期文物风格不统一 | 历史真实性受损 |
| 数据孤岛严重 | 各机构数据无法共享，单机构数据量有限 | 模型泛化能力弱 |
| 修复依据不可追溯 | 修复决策缺乏文献支撑，难以审计 | 公信力存疑 |

### 1.2 政策机遇

- **十四五文物保护科技创新规划**：重点支持文物数字化与智能修复
- **文化数字化战略**：到2035年建成国家文化大数据体系
- **AIGC政策**：鼓励AI赋能传统文化传承

### 1.3 技术机遇

- 扩散模型 (Stable Diffusion, ControlNet) 在图像生成上成熟
- 联邦学习在医疗影像领域已验证隐私保护可行性
- HNSW向量检索在文物相似性匹配上效果显著

## 2. 解决方案

### 2.1 产品定位

**壁画守护者** — 面向文保机构的AI辅助壁画修复SaaS平台

```
┌─────────────────────────────────────────────────────────────┐
│                    Mural Guardian Platform                   │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐ │
│  │  病害智能   │  │  风格一致   │  │   多模态修复         │ │
│  │  检测分割   │→ │  修复建议   │→ │   方案生成           │ │
│  └─────────────┘  └─────────────┘  └─────────────────────┘ │
│         ↑               ↑                   ↑              │
│  ┌─────────────────────────────────────────────────────────┐│
│  │              联邦学习训练层（数据不出域）                  ││
│  │   敦煌研究院 ←→ 各地博物馆 ←→ 研究机构                   ││
│  └─────────────────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────────────────┐│
│  │              区块链审计层（修复可信追溯）                   ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

### 2.2 核心功能

#### F1: 病害智能检测
- **输入**: 壁画高清图片
- **输出**: 病害类型(起甲/酥碱/空鼓等) + 区域分割Mask
- **技术**: SAM + UNet 多标签分割
- **效果**: 识别准确率 > 90%

#### F2: 风格一致修复
- **输入**: 病害区域 + 同期参考文物
- **输出**: 修复方案 + 风格匹配度评分
- **技术**: ControlNet + 壁画风格LoRA微调
- **效果**: 专家认可率 > 70%

#### F3: 相似文物检索
- **输入**: 壁画碎片/局部图
- **输出**: Top-K 相似文物图谱 + 关联文献
- **技术**: ResNet18 特征提取 + HNSW 向量检索
- **效果**: 检索召回率 > 85%

#### F4: 修复方案审计
- **输入**: 修复决策记录
- **输出**: SHA-256 哈希存证 + 可追溯决策链
- **技术**: 区块链哈希链 + IPFS 分布式存储
- **效果**: 全流程可审计、可验证
